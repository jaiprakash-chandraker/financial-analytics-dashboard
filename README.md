# Financial Analytics Dashboard

## Problem
Businesses lack visibility into revenue trends and customer behavior, leading to poor decision-making.

## Objective
To build a dashboard that tracks KPIs, identifies revenue trends, and detects anomalies.

## Tools Used
- SQL
- Power BI

## Key Insights
- Top 10% customers contribute majority of revenue
- Identified revenue decline in specific time periods
- Detected anomalies in transaction patterns
